Free Windows utility

World Time Widget for Windows

Keep USA Eastern, USA PST, Morocco, China, and any custom time zone visible in a compact desktop widget.

Install from GitHub See features

Custom Countries

Add any country, city, or Windows time zone from the built-in settings window.

Compact Desktop Clock

Use compact view for a small always-on-top widget, or full view for seconds, dates, and UTC offsets.

Hide To Tray

Hide the widget when you need a clean desktop, then restore it from the Windows tray.

Install

Clone or download the GitHub repository and run the installer in PowerShell:

powershell -ExecutionPolicy Bypass -File ".\Install-WorldTimeWidget.ps1"
